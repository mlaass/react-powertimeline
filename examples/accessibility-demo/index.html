Accessibility Demo

This example demonstrates PowerTimeline's accessibility features including keyboard navigation, screen reader support, and customization options.

Keyboard Navigation

  • Tab / Shift+Tab - Navigate between interactive elements
  • Arrow Keys - Pan timeline (←→) or navigate lanes (↑↓)
  • + / - - Zoom in/out
  • Home / End - Go to start/end of timeline
  • Space - Select focused item
  • Enter - Activate focused item
  • Escape - Clear selection
Timeline loaded with accessibility features enabled.

Accessibility Features Demonstrated:

  • ARIA Labels: Comprehensive labeling for screen readers
  • Keyboard Navigation: Full keyboard control without mouse dependency
  • Focus Management: Logical focus order and visible focus indicators
  • Live Regions: Dynamic content announcements
  • High Contrast: Enhanced visual contrast for low vision users
  • Reduced Motion: Respects user's motion preferences
  • Screen Reader Support: Compatible with NVDA, JAWS, VoiceOver
  • WCAG 2.1 AA Compliance: Meets accessibility guidelines
Use arrow keys to navigate the timeline. Press Space to select items and Enter to activate them. The timeline contains events, metrics, and time ranges organized in lanes.
