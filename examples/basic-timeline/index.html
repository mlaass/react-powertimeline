Basic Timeline Example

This example demonstrates the basic usage of the PowerTimeline component with simple data.

Features Demonstrated:

  • Multiple Data Types: Events, time ranges, and curve data
  • Pan & Zoom: Mouse wheel to zoom, drag to pan
  • Item Interactions: Click and hover on timeline items
  • Multiple Lanes: Organized data across different lanes

Basic Usage:

import { PowerTimeline } from '@react-powertimeline'; const lanes = [ { id: 'events', height: 60, label: 'System Events' }, { id: 'metrics', height: 100, label: 'Performance' }, { id: 'deployments', height: 40, label: 'Deployments' } ]; const items = [ { id: 'event-1', type: 'event', laneId: 'events', time: new Date('2024-01-01T10:00:00Z'), style: { markerType: 'circle', color: '#dc3545', size: 8 } }, // ... more items ]; <PowerTimeline lanes={lanes} items={items} initialTimeRange={{ start: startTime, end: endTime }} onViewChange={(timeRange) => console.log('View changed:', timeRange)} width={800} height={300} />
